## 用户指令

### 播放音频

指令
: `播放 <单位持续时间(ms)>;<简谱>`

进行简谱到音频的转换。简谱的格式为：

- `1-7` : 对应`do, re, mi, fa, sol, la, ti`, `la = 440Hz`，只对`播放`有效。
- `A-G` : 同上，只对`9播放`有效。
- `b`   : 降半音，写在对应的音前。
- `#`   : 升半音，写在对应的音前。
- <code>`</code> : 高八度，写在对应的音后。
- `.`   : 低八度，写在对应的音后。
- `-`   : 延长前一个音`1`单位。

: ~~你不会真的以为我会写吧~~
